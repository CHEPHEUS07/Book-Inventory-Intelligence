Book Inventory Intelligence: End-to-End Data Analytics & Sentiment Pipeline
📌 Project Overview
This project demonstrates a complete data science lifecycle—from raw data processing to advanced visualization and natural language processing. The goal was to analyze a book inventory dataset to identify pricing trends, stock availability, and the emotional tone of product descriptions to drive better business decisions.  

🚀 Key Features
1. Exploratory Data Analysis (EDA)
Data Cleaning: Leveraged Python to transform raw, messy data into the structured books_cleaned.csv.  

Statistical Profiling: Analyzed variable distributions, handled missing values, and standardized price/tax columns.  

Anomaly Detection: Identified and resolved data inconsistencies to ensure high-quality analysis.  

2. Interactive Business Intelligence (Tableau)
Dynamic Dashboards: Created a comprehensive visual analysis in Tableau (Book_Inventory_analysis.twbx).  

Trend Visualization: Designed charts to track stock levels, category-wise pricing, and tax impacts.  

Data Storytelling: Converted complex inventory metrics into easy-to-understand visual insights for stakeholders.  

3. NLP & Sentiment Analysis
Text Classification: Implemented a Python-based Natural Language Processing (NLP) pipeline to evaluate book descriptions.  

Sentiment Tagging: Classified descriptions into Positive, Neutral, and Negative categories using polarity scoring.  

Market Insights: Correlated book descriptions with ratings to understand how marketing text impacts product perception.  

🛠 Tech Stack
Language: Python 3.x  

Libraries: Pandas, TextBlob, Matplotlib, Seaborn  

BI Tool: Tableau Desktop  

Data Format: CSV / SQL-compatible  

📁 File Structure
scrapped_books.csv: The Scrapped csv file after eb scrapping.

books_cleaned.csv: The primary, high-quality dataset.  

Book_Inventory_analysis.twbx: The interactive Tableau dashboard.  

sentiment_analysis.py: Python script containing the NLP and classification logic.  

books_with_sentiment.csv: The final enriched dataset containing sentiment labels.  

📈 Key Insights
Successfully categorized 60+ book records into distinct sentiment buckets.  

Identified pricing bands (e.g., Luxury vs. Budget) to optimize inventory turnover.  

Visualized the relationship between stock availability and category popularity.
